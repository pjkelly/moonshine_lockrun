= Moonshine Lockrun

=== A plugin for Moonshine[http://github.com/railsmachine/moonshine]

A plugin for installing and managing lockrun.

=== Instructions

* <tt>script/plugin install git://github.com/ACCOUNT/moonshine_lockrun.git</tt>
* Configure settings if needed
    configure(:lockrun => {:foo => true})
* Invoke the recipe(s) in your Moonshine manifest
    recipe :lockrun
